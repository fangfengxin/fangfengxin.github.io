Spring 整合 Spring MVC

整合说明

  • Spring MVC 和 Spring 的整合目标是在 controller 方法中可以调用 service 方法。
  • 但是在启动服务器后，只会加载 Spring MVC 的配置文件，并不会加载 Spring 的配置文件，这样 controller 中无法自动注入 service 对象，进而不能调用 service 方法。
  • 因此需要在启动服务器的时候同时加载 Spring 的配置文件。

Spring 整合 MyBatis 框架

整合说明

  • Spring 整合 MyBatis 的目标是在 service 方法中调用 dao 方法。
  • service 对象已经在 Spring 中配置，需要把 MyBatis 配置产生的代理 dao 对象也配置到 Spring 容器中。
  • 整合思路：把 MyBatis 配置文件中的内容配置到 Spring 配置文件中，同时把 MyBatis 配置文件的内容清掉。
